# tea-fi
Automation Swap Tea Fi

## Features
1. Convert POL to WPOL
2. Wrap WPOL to tPOL
3. Unwrap all tPOL to WPOL
4. Claim Daily Reward
5. Execute options 2, 3, and 4 sequentially

## Setup

1. clone the repo
2. cd to the directory
   ```
   cd tea-fi
   ```
3. install dependency
   ```
   npm install
   ```

4. Modify priv.txt
5. run the node
   ```
   node index.js
   ```
